# CareTaiwan 互動 Demo

線上版本：<https://caretaiwan-demo-yuri.lisarock278496.chatgpt.site>

這是一個自帶示意資料、零後端的互動原型，呈現 CareTaiwan 原本的主要操作流程：

- 看護端：照護紀錄、醫療資訊卡、支出、緊急聯絡與備份
- 家屬端：共享紀錄、健康趨勢、支出確認與 LINE 通知情境
- 所有姓名與健康資料皆為虛構示意內容

index.html 是完整的單檔網站，可直接以瀏覽器開啟，或在這個資料夾啟動任一靜態檔案伺服器。
